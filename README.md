# Opensky Project

This project is a Django-based engineering portal developed as part of coursework. It includes multiple pages for managing and viewing engineering-related information.

## Features

- Home page
- Teams page
- Schedule page
- Messages page
- Organisation page
- Reports page
- PDF export functionality
- Excel export functionality
- GitHub version control

## Technologies Used

- Python
- Django
- HTML
- CSS
- JavaScript
- SQLite
- Git and GitHub

## Project Structure

- `manage.py` – main Django management file
- `opensky/` – project settings and root configuration
- `portal/` – main application (views, templates, static files)

## How to Run the Project

1. Open the project folder in VS Code  
2. Open the terminal  
3. Run:

```bash
python manage.py runserver
